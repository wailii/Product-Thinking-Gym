# Product Thinking Gym (产品思维训练营)

> "这不是温暖的教室，这是无情的对抗竞技场。"

**Product-Thinking-Gym** 是一个通过 **Red Teaming (红队对抗) + 结构化 P&L (损益表) 验证** 来重建产品决策能力的自我训练系统。

## 📂 核心文档

- **📜 [规则说明书 (The Rules)](Product-Thinking-Gym-RULES-V3.md)**: 了解 CTO、VC、User 三大对抗角色以及 "黑话防火墙" 机制。
- **📊 [进度追踪器 (Progress Tracker)](progress/product-tracker.md)**: 记录你的技能树点亮情况和训练进度。
- **📝 [对抗记录模板 (Session Template)](sessions/template.md)**: 每次产生新想法时，复制此模板开始训练。

## 🚀 如何使用

1. **提出想法**: 无论是"AI 数据清洗"还是"新型社交App"。
2. **复制模板**: `cp sessions/template.md sessions/YYYY-MM-DD-MyIdea.md`
3. **自我对抗**: 扮演 CTO、VC、User 三个角色攻击自己，并记录防守过程。
4. **财务验证**: 强制自己算一笔账，画出简易 P&L 表。
5. **复盘归档**: 将结果更新到 `progress/product-tracker.md`。
